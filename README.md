### TP 4 : Hibernate - Configuration d'une Application Java avec MySQL

### Objectif
Ce TP nous a permis de configurer une application Java avec Hibernate et MySQL pour gérer la persistance des données. Vous apprendrez à :
Configurer Hibernate avec Maven et sans oublier la création des entités JPA
et limplementation du patron de conception DAO (Data Access Object) et puis gerer les transactions et les sessions Hibernate
et finalement reussir a tester la persistance des données
 ### Prérequis
Java 8 ou supérieur
Maven 3.6+
MySQL 5.7+ ou MariaDB
IntelliJ IDEA ou Eclipse
